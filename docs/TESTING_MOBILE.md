| Email | Пароль | Описание |
|-------|--------|----------|

Для тестирования админ-панели необходим пользователь с ролью `admin`. Создается через БД или через seed-скрипт.

|------------|---------------|------------|

| iPhone 12/13 | 390px | Мобильный |
| iPhone 12 Pro Max | 428px | Мобильный |
| iPad Mini | 768px | Планшет (sm) |
| iPad Pro | 1024px | Планшет (lg) |
| Desktop | 1280px+ | Десктоп |

### Как тестировать в Chrome DevTools

1. Открыть DevTools (F12 или Cmd+Option+I)
2. Нажать кнопку "Toggle device toolbar" (Cmd+Shift+M)
3. Выбрать устройство или задать размер вручную
4. Обновить страницу

## Общие проверки

### Sidebar (боковое меню)

- [ ] На мобильном скрыт по умолчанию
- [ ] Hamburger кнопка видна в шапке
- [ ] При клике открывается слева
- [ ] Overlay затемняет контент
- [ ] Клик по overlay закрывает меню
- [ ] Escape закрывает меню
- [ ] Переход по ссылке закрывает меню
- [ ] Body не скроллится при открытом меню

### Типографика

- [ ] Заголовки: 20px на мобильном, 24-30px на десктопе
- [ ] Текст: 14px на мобильном, 16px на десктопе
- [ ] Labels: 12px на мобильном, 14px на десктопе

### Отступы

- [ ] Padding карточек: 12-16px на мобильном, 20-24px на десктопе
- [ ] Gap в сетках: 12px на мобильном, 24px на десктопе
- [ ] Margin под заголовками: 16px на мобильном, 32px на десктопе

### Интерактивные элементы

- [ ] Кнопки min-height 44px (touch target)
- [ ] Поля ввода не обрезаются
- [ ] Чекбоксы 18x18px минимум
- [ ] Ссылки кликабельны

## Баги для проверки

### Известные проблемы

1. **Таблицы** - могут быть обрезаны без горизонтального скролла
2. **Dropdown меню** - может выходить за пределы экрана справа
3. **Длинные ID** - должны truncate с ...
4. **Цены** - могут не помещаться (используйте text-xl на мобильном)

### Как репортить баги

```
Страница: /agent/dashboard
Устройство: iPhone SE (375px)
Проблема: Статистика выходит за пределы экрана
Скриншот: [приложить]
```

## Запуск локально

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Backend

```bash
cd backend
pip install -r requirements.txt
uvicorn app.main:app --reload --port 8000
```

### Или через Docker

```bash
docker-compose up -d
```
